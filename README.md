# 📊 Logi-Estimate

**物流・作業現場向け 生産性シミュレーション & 工数見積ツール**

------------------------------------------------------------------------

## 🔍 概要

**Logi-Estimate** は、物流・製造・倉庫業務における作業を\
**WBS（作業分解構成図）** として定義し、\
作業工数・生産性・単位あたり作業時間を
**インタラクティブに可視化・算出** する Web アプリケーションです。

現場ヒアリングで属人化しがちな\
- 「この作業、結局どれくらい時間がかかるのか？」 -
「物量が変わると工数はどう変わるのか？」

といった問いに対し、\
**定量的・再利用可能なモデル** として答えを出すことを目的としています。

------------------------------------------------------------------------

## 🎯 想定ユースケース

-   物流センターの作業工数見積
-   業務改善（Before / After 比較）
-   人員計画・生産性シミュレーション
-   新拠点・新ライン立ち上げ時の工数算定
-   現場ヒアリング結果の構造化・可視化

------------------------------------------------------------------------

## ✨ 主な機能

### 🧩 WBS（作業分解構成図）設計

-   グループタスク / 作業 / 搬送作業 の階層構造
-   ドラッグ & ドロップによる直感的な編集
-   繰り返し回数・子タスク集計に対応

### ⏱️ 工数・生産性計算

-   標準時間 × 物量 × 能率 × 余裕率 に基づく工数算出
-   単位あたり作業時間を自動計算
-   合計作業時間をリアルタイム更新

### 📦 単位変換ロジック

-   ピース / 箱 / パレット を相互変換
-   プロジェクト共通設定 + タスク個別設定
-   現場ごとの換算ルールに対応

### 🔄 履歴管理

-   Undo / Redo 対応（最大50ステップ）

### 💾 データ永続化

-   LocalStorage によるプロジェクト保存
-   自動保存機能

### 📥📤 データ入出力

-   CSVインポート
-   Excelエクスポート

### 🔎 検索・フィルタリング

-   作業名・カテゴリ検索
-   単位別フィルタ

------------------------------------------------------------------------

## 🖥️ 技術スタック

-   HTML5 / CSS3
-   Vanilla JavaScript
-   LocalStorage / JSON

------------------------------------------------------------------------

## 🚀 使い方

1.  index.html をブラウザで開く\
2.  タスクを追加してWBSを構築\
3.  工数・生産性を確認\
4.  必要に応じてExcel出力
